# TCATemplate

A minimal iOS-only template app using The Composable Architecture (TCA), modeled after the `try-swift-tokyo` split of:

- `App/`: iOS application target (generated by XcodeGen)
- `iOS/`: Swift package that contains TCA features and reducer tests

## Features

- Counter tab (`CounterFeature`)
- Navigate tab (`SecondTabFeature`) with one detail screen transition
- Reducer test example (`CounterFeatureTests`)

## Setup

1. Generate the Xcode project.

```bash
cd App
xcodegen generate
```

2. Open `App/TCATemplate.xcodeproj` in Xcode and run the app.

## Test

```bash
swift test --package-path iOS
```
